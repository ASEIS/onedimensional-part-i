\section{Equivalent linear method}
\subsection{Modulus degradation and damping relationship}

A number of modulus reduction and damping relationships are available for use in equivalent-linear analyses \citp[e.g.,][]{Idriss1970,Vucetic1991,Ishibashi1993,Darendeli2001,Zhang2005}. \citet{Kaklamanos2013} used the \citet{Zhang2005} relationships for equivalent site response calculation.
